    Responsive website: HTML and css for the layout.

    This is a mock up. A flower company thats donates a portion of its profits to endangered animals. I came up with the concept when i was drawing endangered animals. It is a website to create awareness for endangered animals. I drew the tiger and the flower. I used photoshop to clean it up and add color. I put links to endangered animal foundations. I used vanilla javascript to make the fade out affect on the text.

    A minimalist website thats works for desktop and mobile.

    I learned how to apply css to a html file witout using front-end frameworks.
